What is BirckEngine

BrickEngine is a 2d engine build in GameMaker: studio 1.4 used for brickbuidling games in this engine you can build Levels with Parts imagine it as virtual 2d lego bricks for example fun fact: This engine started as a roblox fangame called "Roblox2D" or "RBLX2D"

Part Object

a Part Object (obj_part) is the building brick object of BrickEngine used to build the sctructure of a Level a Part Object includes the following propeties:

  • SizeX
  • SizeY
  • X
  • Y
  • ColorId
  • CanCollide

to create a part on the middle of the screen, use the following example code below:

///THIS CREATES A PART instance_create(view_xview+(view_wview/2),view_yview+(view_hview/2),obj_part)
Levels

a Level is the currentlly loaded map/build on BrickEngine these can be saved in the .blvl format in json formatting a Level can include these following propeties:

  • SkyColor
  • LevelName
